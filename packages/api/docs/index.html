PageSnap API Documentation

POST /api/v1/convert

Converts one or more webpages to the specified formats.

Request Body

{
  "urls": ["https://example.com"],
  "formats": ["png", "jpg", "pdf"],
  "clip": { "x": 0, "y": 0, "width": 1280, "height": 800 },
  "watermark": { ... },
  "callbackUrl": "https://your-webhook-url.com"
}

Parameters

  • urls (array, required): An array of URL strings.
  • formats (array, optional): Overrides the default formats.
  • clip (object, optional): Defines a capture region.
  • watermark (object, optional): Watermark configuration.
  • callbackUrl (string, optional): If provided, the request is processed asynchronously.

POST /api/v1/render/html

Renders raw HTML content as an image.

Request Body

{
  "html": "<h1>Hello, World!</h1>",
  "formats": ["png"],
  ...
}

GET /render/v1/direct

Generates a direct image URL for embedding.

Query Parameters

  • url (string, required): The URL to capture.
  • signature (string, required): HMAC signature.
  • ... (other screenshot options)
